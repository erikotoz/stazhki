# Стажки в расчёте

Многопользовательское приложение для деления расходов: вход по нику/паролю,
у каждого — сколько ему должны и сколько должен он (с раскрывашкой «кто и за что»),
граф расчётов с тобой в центре.

## Возможности

- **Личный баланс**: «Вам должны» / «Вы должны», раскрывается до людей и до конкретных трат.
- **Переводы**: в разделе «Кому должны вы» жмёшь «Я перевёл(а) …» — долг гасится сразу,
  графы пересчитываются. Получатель видит **уведомление** (колокольчик) и нажимает
  «Подтвердить» или «Не получил» (тогда долг возвращается, а тебе приходит уведомление).
- **Граф расчётов** — ты в центре, остальные по кругу; зелёный/красный = в плюсе/в минусе.
- **Траты**: фильтр «Все / Мои» (только операции с твоим участием).
- Светлая/тёмная тема, два режима свода (по парам / минимум переводов).

## Как запустить

```bash
cd ~/pay-app
python3 server.py
```

Открой в браузере: **http://localhost:8000**

Зависимостей нет — только стандартный Python 3. Данные хранятся в файле `cash.db`
(SQLite) рядом со скриптом.

## Чтобы зашли друзья (по той же Wi-Fi)

Сервер слушает все адреса, поэтому с телефона в той же сети открывайте:

```
http://192.168.1.2:8000
```

(192.168.1.2 — IP этого компьютера сейчас; если сеть сменится, посмотри командой
`ipconfig getifaddr en0`). Компьютер с сервером должен быть включён.

Выход в «большой интернет» (чтобы заходили откуда угодно) — отдельный шаг (хостинг),
сделаем позже.

## Аккаунты

- Регистрация — на экране входа вкладка «Регистрация» (ник от 2 символов, пароль от 3).
- Сейчас в базе уже есть тестовые аккаунты с примером из переписки:
  **egor / erik / maxim**, пароль у всех `123`. Войди как `egor` — увидишь готовый расчёт.

## Сбросить все данные

Останови сервер (Ctrl+C) и удали базу:

```bash
rm ~/pay-app/cash.db ~/pay-app/cash.db-wal ~/pay-app/cash.db-shm
```

## Что где лежит

| Файл | Назначение |
|------|------------|
| `server.py` | сервер: API + раздача фронта, SQLite, вход по паролю |
| `web/index.html` | разметка и стили |
| `web/app.jsx` | основная логика интерфейса (вход, личный баланс, разбивка) |
| `web/graph.jsx` | граф расчётов (центр — текущий пользователь) |
| `web/sheet.jsx` | форма добавления/редактирования траты |
| `web/compute.js` | математика балансов и минимальных переводов |
| `web/icons.jsx` | иконки и категории |
| `cash.db` | база данных (создаётся автоматически) |
| `prototype-v1.html` | первый локальный прототип (одним файлом), не используется сервером |
